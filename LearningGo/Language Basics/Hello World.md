```go
package main  
  
import "fmt"  
  
func main() {  
   fmt.Println("Hello, World!")  
}
```

To run do
`go run .` : To run a package, it can have only one file with `package main`
`go run helloWorld.go`: To run a go file which must have `package main`

For build
`go build .`

To execute build
`./ProjectOrDirectoryName`
